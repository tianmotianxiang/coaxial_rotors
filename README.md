# 共轴式双旋翼
> 两个旋翼上下排列在同一个轴上，转向相反，两副旋翼产生的扭矩在航向不变的飞行状态下相互平衡，通过所谓的上下旋翼总距差动产生不平衡扭矩可实现航向操纵，共轴双旋翼在直升机的飞行中，既是升力面又是纵横向和航向的操纵面。
